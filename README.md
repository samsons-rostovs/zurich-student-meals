# Zurich Student Meals

A lightweight Python tool that aggregates daily ETH/UZH mensa meals and finds the cheapest student lunch options in Zurich.

## Features

- Scrapes daily mensa menus
- Extracts student prices
- Sorts meals by price
- Built using Python, requests, and reverse-engineered Next.js data payloads

## Example Output

6.1 CHF | garden | SWEET & SOUR...
7.5 CHF | butcher | RIND GHACKETS...
9.0 CHF | pure asia | SCHNITZEL TON KATSU...

## Motivation

Built as a practical student-focused automation project and engineering exercise in web scraping and data extraction.
